## 概要

長さ $N$ の配列 $S$ に対して，$S$ と $S[i:]$ の最長共通接頭辞 (LCP: Longest Common Prefix) の長さを求める．

アルゴリズムの計算量は $\mathcal{O}(N)$ ．
